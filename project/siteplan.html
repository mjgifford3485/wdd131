ttrpgprimer.com

I chose this name because this website is a primer (a small introductory book or piece of writing) on tabletop roleplaying games(ttrpg).

Site Purpose

The purpose of this website is to provide an explanation of what tabletop roleplaying games are along with a description of some of the more popular games and resources for finding more information.
